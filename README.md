Matchjs-mobile
==========

Mobile web client for MatchJS.

Stack
===========

- React
- GraphQL and Relay
- RxJS
- Immutable.js
- AMP HTML
- Flexbox + LESS
- Webpack
- Koa
- Node.js/IO.js
- MongoDB with Mongoose
- Babel

Starting
===========

- `node --harmony index.js`

Matchjs-mobile
==========

Mobile web client for MatchJS.

Stack
===========

- Front-end
  - react
  - react-router
  - relay
  - flexbox
  - less

- Server
  - graphql
  - koa

- Build
 - babel
 - webpack
 - eslint

- Tests
 - karma, phantomjs, chrome
 - mocha, sinon, chai

- TBD
 - RxJS
 - AMP HTML
 - MongoDB with Mongoose
 - Immutable.js

Starting
===========

- `npm start`

===========
